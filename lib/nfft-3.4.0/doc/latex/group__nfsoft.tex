\subsection{Detailed Description}
This module implements nonuniform fast S\-O(3) Fourier transforms. In the following, we abbreviate the term \char`\"{}nonuniform fast S\-O(3) Fourier
transform\char`\"{} by N\-F\-S\-O\-F\-T.
